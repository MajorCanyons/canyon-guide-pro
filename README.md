# MajorCanyons GitHub Pages

## Veröffentlichung

1. Auf GitHub ein neues öffentliches Repository anlegen.
2. Empfohlener Name:
   - `majorcanyons.github.io`, wenn der GitHub-Benutzername `majorcanyons` lautet.
   - Andernfalls z. B. `canyon-guide-pro`.
3. Alle Dateien aus diesem Ordner in das Repository hochladen.
4. Repository öffnen: Settings → Pages.
5. Unter “Build and deployment” auswählen:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
6. Speichern und einige Minuten warten.
7. Die veröffentlichte URL in Google Play bei „Datenschutzerklärung“ eintragen:
   - Benutzer-/Organisationsseite: https://BENUTZERNAME.github.io/datenschutz.html
   - Projektseite: https://BENUTZERNAME.github.io/REPOSITORY/datenschutz.html

## Vor Veröffentlichung prüfen

Die Datenschutzerklärung basiert auf dem derzeit beschriebenen Funktionsumfang:
- lokale Datenspeicherung
- optionaler Standortzugriff
- optionale OpenCanyon-Verbindung
- keine Werbung
- keine eigenen Analyse- oder Trackingdienste
- PDF-Export nur auf Nutzeraktion

Falls die App technisch davon abweicht, müssen Datenschutzerklärung und Google-Play-Datensicherheitsformular angepasst werden.
